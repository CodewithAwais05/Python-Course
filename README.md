# Python Full Course 

A complete roadmap covering Python fundamentals to Object-Oriented Programming, organized chapter-wise.

---

## Chapter 1: Variables & Data Types
- Understanding variables and how Python stores values
- Naming rules and conventions for variables
- Core data types: `int`, `float`, `str`, `bool`
- Type checking (`type()`) and type conversion/casting
- Arithmetic, comparison, and logical operators
- Taking user input and handling its default string type

## Chapter 2: Strings & Conditional Statements
- String creation, indexing, and slicing
- Common string methods (`upper()`, `lower()`, `strip()`, `split()`, `replace()`, `join()`, etc.)
- String immutability
- Conditional logic using `if`, `elif`, `else`
- Combining conditions with `and`, `or`, `not`
- Indentation rules and nested conditionals

## Chapter 3: Lists & Tuples
- Creating and working with lists (ordered, mutable collections)
- List methods: `append()`, `insert()`, `remove()`, `pop()`, `sort()`, `reverse()`
- Creating and working with tuples (ordered, immutable collections)
- Differences between lists and tuples
- Indexing, slicing, and membership checks (`in`)

## Chapter 4: Dictionary & Sets
- Key-value pairs using dictionaries
- Dictionary methods: `keys()`, `values()`, `items()`, `get()`, `update()`
- Adding, updating, and deleting dictionary entries
- Sets as unordered, unique-value collections
- Set operations: union, intersection, difference
- Use cases for dictionaries vs sets

## Chapter 5: Loops in Python
- `while` loops and their use cases
- `for` loops and iterating over sequences (lists, strings, ranges)
- `range()` function and its parameters
- Loop control statements: `break`, `continue`, `pass`
- Nested loops
- Common patterns: counting, accumulating, searching

## Chapter 6: Functions & Recursion
- Defining and calling functions with `def`
- Parameters, arguments, and default values
- Return values vs `print()` inside functions
- Variable scope (local vs global)
- Recursive functions and base cases
- Difference between iterative and recursive approaches

## Chapter 7: File Input/Output
- Opening and closing files using `open()` and `close()`
- File modes: read (`r`), write (`w`), append (`a`)
- Reading file content: `read()`, `readline()`, `readlines()`
- Writing and appending data to files
- Using `with` statement for safe file handling
- Working with file paths

## Chapter 8: OOPS — Object Oriented Programming (Part 1)
- Core OOP concepts: classes and objects
- Defining classes and creating instances
- The `__init__` constructor method
- Instance attributes and methods
- The `self` keyword explained
- Difference between class-level and instance-level data

## Chapter 9: OOPS — Object Oriented Programming (Part 2)
- Inheritance and creating subclasses
- Method overriding
- Encapsulation and access modifiers (public, protected, private conventions)
- Polymorphism in Python
- Special/dunder methods (e.g., `__str__`, `__repr__`)
- Building more complex, real-world class structures
